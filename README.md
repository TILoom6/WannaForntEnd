## Getting Start

### install

```sh
npm i
```

or

```
yarn install
```

### Activate DevServer

```sh
npm start
```

### Build

```sh
npm run build
```

### Test

```sh
npm run test
```

### Lint

```sh
npm run lint
```

### Type Check

```sh
npm run flow
```
